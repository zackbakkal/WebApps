HTML5 Document

HTML stands for Hyper Text Markup Language. It describes the structure of Web pages using markup. HTML elements are the building blocks of HTML pages. The elements are represented by tags. These tags label pieces of content such as "heading", "paragraph", "table", and so on.

Each element has a start TAG and an end TAG

A start tag is written as follow <elementName>, while the end tag is written as follow </elementName>.

Browsers do not display the HTML tags, but use them to render the content of the page

Example:

<!DOCTYPE html>

<html>

<head>

<title>Page Title Here</title>

</head>

<body>

<h1>H1 Heading</h1>

<h2>H2 Heading</h2>

<h3>H3 Heading</h3>

<p>This is a Paragraph</p>

</body>

</html>

Tags definitions

  • The <!DOCTYPE html> declaration defines this document to be HTML5
  • The <html> element is the root element of an HTML page
  • The <head> element contains meta information about the document
  • The <title> element specifies a title for the document
  • The <body> element contains the visible page content
  • The <h1> element defines a large heading
    • Heading elements goes from h1 to h6, h1 is the biggest and h6 is the smallest
  • The <p> element defines a paragraph

img Element

You can improve your website design and appearance by including images.

In HTML, images are defined with the <img> tag

The <img> tag is empty, it contains attributes such as the src, and alt attribute, and does not have a closing tag.

The src attribute specifies the URL of the image, while the alt provides an alternate test for an image. The alt attribute is useful in case the user cannot view the image. The alt text describes the image.

Example:

<!DOCTYPE html>

<html>

<head>

<title>HTML Images</title>

</head>

<body>

<img src="images/birds.jpg" alt="Wild Birds">

</body>

</html>

Image size

You can specify the image's width and height by using the width and height attributes. The width and height attributes always defines the width and height of the image in pixels.

<!DOCTYPE html>

<html>

<head>

<title>HTML Images</title>

</head>

<body>

<img src="images/birds.jpg" alt="Wild Birds" width="200" heigth="300">

</body>

</html>

You can also specify the image's width and height by using the style attribute.

<!DOCTYPE html>

<html>

<head>

<title>HTML Images</title>

</head>

<body>

<img src="images/birds.jpg" alt="Wild Birds" style="width:200px;heigth:300px;">

</body>

</html>

Note: Using the style attribue prevents style sheets from changing the size of images.
